# 💰 Student Expense Tracker

## 📌 Overview

The **Student Expense Tracker** is a simple web application built using **Python, Streamlit, and Pandas** that helps students record, track, and analyze their daily expenses.

Students often struggle with financial management because they do not monitor where their money is spent. This project provides a simple interface where users can add expenses and visualize their spending patterns using charts.

---

## 🎯 Problem Statement

Many students spend money daily on food, travel, entertainment, and shopping but fail to track these expenses. Without proper tracking, it becomes difficult to manage budgets and control unnecessary spending.

This project aims to solve that problem by creating an easy-to-use expense tracking system.

---

## 🚀 Features

* Add new expenses through a sidebar form
* Categorize expenses (Food, Travel, Shopping, etc.)
* Store expense data in a CSV file
* Display all expenses in a table
* Generate category-wise expense charts
* Show spending trends over time

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/yourusername/student-expense-tracker.git
```

2. Navigate to the project directory:

```
cd student-expense-tracker
```

3. Install required libraries:

```
pip install streamlit pandas
```

---

## ▶️ Running the Application

Run the following command in the terminal:

```
streamlit run app.py
```

After running the command, the application will open in your browser at:

```
http://localhost:8501
```
